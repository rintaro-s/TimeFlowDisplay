Mi Dashboard

Ambient productivity hub for wall-mounted Android tablets

壁掛けタブレットを、あなただけのワークステーションに。

Mi Dashboard は Jetpack Compose で作られた常設ダッシュボード。
時計・カレンダー・メモ・タイマー・AI チャット・Webhook ボタンなど、
日々のワークフローに必要なウィジェットを 1 つの画面に集約します。

レイアウトはドラッグ&ドロップで編集し、Room と DataStore に保存。
暗転スケジュールや壁紙ディムで、昼夜を問わず視認性の高い情報スペースを実現します。

ドラッグ&ドロップ編集

時計、メモ、タイマー、画像などのウィジェットを自由に配置。
サイズや不透明度も即座に調整できます。

ローカル AI & Webhook

LMStudio 互換 API と接続して AI チャットを実現。
Webhook ボタンで Discord などの自動化をトリガー。
